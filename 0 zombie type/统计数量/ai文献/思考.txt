不过，可能更简单的是寻找递推式的特征方程。

这个递推式是一个线性递推关系，其齐次递推式为：

f(k) - f(k−1) + p^n (1−p) f(k−n−1) =0

对应的特征方程是什么呢？

通常，线性递推关系的特征方程是通过假设解为r^k的形式代入齐次方程得到的。

即，假设解为f(k) = r^k，代入齐次方程：

r^k - r^{k−1} + p^n (1−p) r^{k−n−1} =0

两边除以r^{k−n−1}：

r^{n+1} - r^{n} + p^n (1−p) =0

所以特征方程为：

r^{n+1} - r^{n} + p^n (1−p) =0

即，

r^n (r -1) + p^n (1−p) =0

这个方程可能很难解析求解，不过当M很大时，主导的根可能是接近1的根，或者其他根绝对值小于1，这样当k很大时，解主要由最大的根决定。

不过或许可以近似求解这个特征方程，当M很大的时候，假设解的主要部分是指数衰减的，即f(k) ≈ C λ^k，其中λ是特征方程的最大根（绝对值小于1），这样概率1 - f(M) ≈1 - C λ^M ≈1，当λ <1且M很大时。不过这可能不准确，需要更仔细的分析。

或者，当M远大于n时，可以用泊松近似或者其他近似方法，假设事件“出现连续n次成功”的发生次数近似服从泊松分布，这样概率约为1 - e^{-λ}，其中λ是期望出现的次数。

这时候，计算在M次试验中，出现连续n次成功的期望次数，然后用泊松近似计算至少发生一次的概率。

当M远大于n时，这可能是一个有效的近似方法，特别是当p不是特别大的时候，使得事件之间近似独立。

连续n次成功可以发生的起始位置有M -n +1个位置，每个位置出现连续n次成功的概率是p^n，因此期望次数为 (M -n +1) p^n ≈ M p^n（因为M远大于n）

根据泊松近似，当概率较小，且试验次数较多时，事件发生的次数近似服从泊松分布，参数λ = M p^n。因此，至少发生一次的概率约为1 - e^{- M p^n}。

不过这可能是一个近似解，当M p^n 不太大的时候适用。但题目中并没有给出具体的参数范围，只是说M远大于n，所以可能需要更多的条件。

不过当M非常大的时候，如果p是固定的，且n也是固定的，那么当M趋向于无穷时，这个概率趋近于1，因为只要试验次数足够多，几乎肯定会发生连续n次成功。

不过题目中可能希望得到一个更精确的表达式，或者一个关于M的指数型表达式。

不过原来的递推式可能对于大的M来说，计算起来比较困难，所以可能需要寻找一个指数型的近似解，例如假设f(k) ≈ C λ^k，代入递推式：

λ^{k} = λ^{k -1} - p^n (1 - p) λ^{k -n -1}

两边除以λ^{k -n -1}，得到：

λ^{n +1} = λ^{n} - p^n (1 - p)

即，

λ^{n +1} - λ^{n} + p^n (1 - p) =0

也就是特征方程，和之前的一样。这时候，最大的根λ <1，因为当λ=1时，方程左边为1 -1 + p^n (1 -p) = p^n (1 -p) >0，而当λ趋近于0时，左边趋近于0 -0 + p^n (1 -p) >0。所以可能最大的根在(0,1)之间。

因此，当k很大时，f(k) ≈ C λ^k，其中λ是特征方程的最大根，因此所求概率为1 - f(M) ≈1 - C λ^M.

不过要找到λ的具体表达式可能比较困难，但或许当M远大于n时，可以用某种近似。

(1 - ε)^{n +1} - (1 - ε)^n + p^n (1 - p)=0

对于一般的n，特征方程为r^{n+1} - r^n + p^n (1 - p)=0

令r =1 - ε，其中ε是一个小量，代入方程：

(1 - ε)^{n +1} - (1 - ε)^n + p^n (1 - p)=0

展开到二阶：

(1 - (n +1)ε + (n+1)n ε^2/2) - (1 -n ε +n(n-1) ε^2/2) + p^n (1 - p) ≈0

化简：

[1 - (n+1)ε + (n+1)n ε^2/2] - [1 -n ε +n(n-1) ε^2/2] + p^n (1 -p) ≈0

- (n+1)ε + (n+1)n ε^2/2 +n ε -n(n-1) ε^2/2 + p^n (1 -p) ≈0

合并同类项：

[- (n+1)ε +n ε] + [ (n+1)n ε^2/2 -n(n-1) ε^2/2 ] + p^n (1 -p) ≈0

即：

-ε + [ (n^2 +n -n^2 +n)/2 ) ε^2 ] + p^n (1 -p) ≈0

进一步化简：

-ε + (2n)/2 ε^2 + p^n (1 -p) ≈0

-ε +n ε^2 + p^n (1 -p) ≈0

不过当p^n (1 -p)很小时，即当n较大或p较小时，sqrt(1 -4n p^n (1 -p)) ≈1 -2n p^n (1 -p), 因此，
