
# 阳光模型
## 模型内容
- 因子1
- 因子2
- 系数

如果
- 因子1为空，因子2为空，为截距，1*1
- 因子1非空，因子2为空，为单因子效应，x*1
- 因子1非空，因子2非空，为双因子效应，x*y


# 设置
## 通用设置
- 场地名称
- 变量取值
- 密度阈值线
- userid，modeid，level
- 冲关长度

## 功能1:随机种子冲关

### 效果
随机模拟冲关，输出“破阵率”
按最终阳光排序，输出最难种子（需要id与level），到文件

### 设置
- 模拟次数（多线程）
- 阳光初值，破阵阳光
- 输出种子个数

## 功能2:指定种子冲关
- 一列种子

## 功能3:寻找难度种子

### 效果
遍历种子，滑动窗口，输出分数+种子的csv

### 设置
- 阳光模型系数必须为整数

- 查询种子初值，查询种子
- 阳光初值，破阵阳光
