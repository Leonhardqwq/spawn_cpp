# 种子破解器使用说明

## 配置方法

1. 在 `json` 文件中填写参数，保存后运行可执行文件。
2. 参数说明如下：

| 参数名       | 类型      | 说明                                  |
| ----------- | --------- | ------------------------------------- |
| scene       | uint32    | 场地（0：DE/MGE/AQE，1：NE，2：PE/FE，3：RE/ME）|
| start_flag  | uint32    | 起始已完成f数（偶数，且不小于16）        |
| user_id     | uint32    | 用户id，例如game1_13中的1               |
| mode_id     | uint32    | 模式id，例如game1_13中的13              |
| masks       | uint32[]  | 出怪类型掩码（数组）                     |

3. 掩码数组需要使用`masks_cal.xlsx`文件来计算，在excel中依次填入连续多次选卡的出怪类型（出对应僵尸填1，否则不填），将MASK一行数字以英文逗号为间隔填入`"masks"`后的`[]`中，一般来说破解种子需要的选卡次数不少于3。


## 运行说明

- 程序运行结束后，将输出：
  - 一大串展示进度但是意义不明的数字
  - 是否成功破解种子，如果成功将输出`FOUND`，否则将输出`NOT FOUND`，此时需要检查掩码计算器中的输入是否正确
- 同路径下会生成 `output.csv` 文件，记录破解得到的所有种子，如果成功破解但破解的种子不唯一，说明该种子需要更多次选卡的出怪信息才能破解
